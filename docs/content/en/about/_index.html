QViz.SDK (Quality Visualizations Software Development Kit) is an open source and free to use set of Class libraries on supported languages to help the Developers who are integrating the QViz.io Product into their Automation Testing Projects using those languages.

{{< /blocks/cover >}} {{% blocks/lead %}}

QViz.SDK Features


  • Open Source and completely FREE to use in any Projects.
  • Fully cross-platform compatible (works on Windows, Mac OS and Linux/Unix).
  • Fully cross-language compatible (almost same Object names across different languages).
  • Supports headless GUI testing using Chrome, Firefox and PhantomJS (for CI/CD Servers).
  • Supports fully automated loop-based testing of unlimited number of GUI and API test cases without individual functions.
  • Fully cross-deploy compatible (works on Cloud Apps, Virtual Machines and Containers).
  • Faster integration to streamline development
  • Very well documented and supports consistent logging across different languages.
  • User Guide and API Docs ready

About QViz.io Product


How much and how early do you know about your product quality issues?

QViz enables enhanced testing visibility and product quality insights by integrating seamlessly across your project workflow.

Intelligent Quality Management with enhanced visibility

  • Enable product teams with quality insights
  • Get more from your present testing tools
  • Jumpstart end-to-end quality governance
  • Integrate seamlessly across project workflow
  • Access historical & current product health statistics

Visual dashboard with detailed statistics

Dashboard designed for product and program teams help you get business visibility into testing processes in order to reduce DEV-QA Cycle time and monitor ROI from testing infrastructure

Historical trends and analytics

Centralized results and data from all teams helps you discover the product quality issues earlier with confidence in order to improve product quality and User Experience

Testing Integration and automation

Support for most used testing tools helps you automate testing with ease and gain reliable visibility into every layer of your feature including UI, API, Mobile, Databases, Security, Performance through your current tools

API integration with other Workflow and Deployment tools

Integration with CI/CD, project management and testing tools helps you optimize testing process to release faster and better in order to improve customer satisfaction
